# App Gym (Buscador Personal Trainer)

App web para registrar entrenamientos en el gimnasio con una experiencia guiada paso a paso.

## Prerequisites

- Node 20+ (recomendado Node 22 si usas `.nvmrc` más adelante)
- npm

## Setup

```bash
npm ci
```

## Dev

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview (serve build)

```bash
npm run preview
```

## Test

```bash
npx vitest run
```

> Nota: el deploy (GitHub Pages / hosting estático equivalente) se documenta en fases posteriores.
